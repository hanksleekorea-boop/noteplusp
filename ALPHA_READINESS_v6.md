현재 정본: `노트앱_v8.html`

앱 SHA-256: `931DCD4EED4FE128A3A549D018795BEF130442B2FAD00A41E35853D3DE24F460`

서비스 워커: `noteplusp-v8-shell-1`

이 문서는 공개 v8 최신 회귀 10종과 실제 Samsung Android 전수 재검증 이후의 상태 정본이다. v5 이하 대시보드는 당시 증거로 보존한다.

## 1. 현재 사실과 판정

- 공개 v8은 HTTP 200, 238,434바이트이며 로컬·공개 SHA-256이 일치한다.
- 공개 Microsoft Edge에서 현재 v8 대상 회귀 10종을 모두 통과했다. 과거 v1~v3 고정 검사는 이전 앱 버전의 불변 증거이므로 최신 전수 집계에서 제외했다.
- Samsung SM-G996N(Android 15, Chrome 150.0.7871.124)에서 IDB 노트 저장·재접속, ENEX PDF Blob 재접속, 서비스 워커 제어, 오프라인 재진입, 모바일 설치 경로와 수평 넘침 없음이 통과했다.
- 휴대폰 시험 전 상태는 활성 노트 1개·휴지통 0개였다. 남아 있던 과거 QA 시험 자료 2개와 첨부 1개를 먼저 감사·제거했고, 이번 시험 자료도 모두 제거했다. 종료 시 QA 노트 0개, ENEX 시험 노트 0개, 시작 상태 서명 일치다.
- 저장 완전 차단 시 입력 유지·무정지·세션 한정 고지, IDB 단독 차단 시 localStorage 보조 저장·첨부 세션 한정 고지를 재확인했다.
- ENEX·JSON·휴지통·붙여넣기 외부 HTML은 저장 전 화이트리스트를 통과하며 외부 요청과 코드 실행은 0건이었다.
- 현재 자동·실기기 범위 미해결 P0·P1은 0건이다. `/favicon.ico` 404는 P2다.
- 실제 한국어 사용자 ENEX 1세트, iPhone, 비개발자 파일럿 3~5명은 아직 외부 의존성이다.

## 2. 제한 알파 관문

| 관문 | 상태 | 근거와 남은 일 |
|---|---|---|
| 1. 누구나 여는 공개 릴리스 | 통과 | 공개 v8 HTTP 200, 로컬·공개 SHA 일치 |
| 2. 실제 모바일 핵심 여정 | 통과 | 실제 Samsung Chrome의 저장·재접속·ENEX PDF·오프라인·설치 경로·원상복구 |
| 3. 저장·데이터 안전 | 통과 | IDB·보조 저장·전체 차단·schema 4→5·외부 HTML·빈 ENEX·백업 복원 회귀 |
| 4. 비개발자 파일럿 3~5명 | 미통과 | 실제 참여자 필요 |
| 5. 종료 품질 판정 | 부분 | 현재 범위 P0/P1=0. 실제 ENEX·파일럿 결과와 남은 한계 종료 보고 필요 |

**현재 공식 진척도: 통과 3/5, 부분 통과 1, 미통과 1. 공식 통과율 60%.**

## 7. 증거와 제한

- 실행 하네스: `tests/test_physical_android_cdp_v5.mjs`와 공개 v8 회귀 10종
- 기계 판정: `artifacts/physical_android_alpha_v5.json`
- 시각 증거: `artifacts/physical_android_alpha_v5.png`
- 휴대폰 Chrome에는 기존 탭이 수백 개 있어 브라우저 전체 대상 열거가 불안정했다. 한 기존 HTTP 탭만 직접 연결해 시험하고 `finally`에서 원래 URL을 복원했다.
- 휴대폰 전면 앱이 YouTube·알림창으로 바뀌어 일반 터치 자동화가 불안정했다. 데이터·저장·오프라인 판정은 실제 Android Chrome 엔진에서 직접 실행했고, 완료 후 ADB 화면 캡처로 실제 모바일 UI를 별도 확인했다.
- 앱 코드는 변경하지 않았다. 현재 정본과 공개 주소는 v8 그대로다.
