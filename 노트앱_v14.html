Evernote 모든 노트·첨부 가져오기

한 화면에서 한 행동만 따라 하세요. Evernote에서는 노트북별로 저장하고, 노트플러스P에서는 폴더를 한 번만 선택합니다.

먼저 이것만 확인하세요

이 작업은 PC에서 계속해야 합니다Evernote ENEX 내보내기는 Windows 또는 Mac 데스크톱 앱에서만 됩니다. 웹·휴대폰에서는 내보낼 수 없습니다.
Evernote에서는 노트북별로 저장 → 노트플러스P에서는 폴더를 딱 한 번 선택Evernote에는 계정 전체를 한 파일로 내보내는 공식 버튼이 없습니다. Notes에서 개별 선택하면 최대 100개 제한이 있으므로 노트북 전체를 하나씩 저장합니다.

Evernote 공식 제한 확인

PC와 백업 폴더를 준비하세요

이 단계에서는 내보내지 않습니다. Evernote 동기화와 저장 위치만 준비합니다.

1
Evernote 데스크톱 앱을 열고 올바른 계정으로 로그인

왼쪽 노트북 목록과 최근 노트가 모두 보일 때까지 동기화를 기다립니다.

2
바탕화면에 ‘Evernote 전체 백업’ 폴더 만들기

앞으로 내보내는 모든 `.enex` 파일을 이 폴더 하나에 저장합니다.

완료 모습: 빈 `Evernote 전체 백업` 폴더가 열려 있고 Evernote의 Notebooks(노트북) 목록이 보입니다.

고급 선택: PC에서 모든 노트북 자동 백업

  • 노트북이 매우 많다면 검증된 오픈소스 `evernote-backup` 도우미로 노트북별 ENEX를 자동 생성할 수 있습니다.
  • Evernote 공식 도구가 아니며 작업·알림은 공개 API 제한으로 완전하지 않을 수 있습니다. 공식 노트북별 방식이 기본 권장 경로입니다.
PC 전체 자동 백업 도우미 받기

실제 화면의 빨간 점선만 따라 하세요

Notebooks(노트북)에서 노트북 하나를 열어 ENEX로 저장합니다. Notes(노트) 전체 선택은 사용하지 않습니다.

1
Notebooks → 노트북의 … → Export notebook…

점 3개가 없으면 노트북 이름을 마우스 오른쪽 버튼으로 누르고 `Export notebook…` 또는 `Export…`를 고릅니다.

2
ENEX + 모든 속성 → Export

`ENEX format (.enex)`를 선택하고 보이는 속성을 모두 체크합니다. 노트플러스P에는 고정 파일 크기 제한이 없습니다. Evernote가 여러 파일로 나누면 생성된 ENEX를 모두 같은 폴더에 보관하세요.

피해야 할 경로: Notes → 전체 선택 또는 File → Export Notes…는 최대 100개 제한에 걸릴 수 있습니다. PDF 형식은 첨부 원본 보존용으로 사용하지 마세요.

화면 예시는 2026-07-19에 공식 절차와 다시 대조했습니다. 설치 버전에 따라 점 3개의 위치가 다르면 노트북 이름을 우클릭하세요.

모든 노트북을 빠짐없이 반복하세요

노트북 이름을 쉼표나 줄바꿈으로 붙여 넣어 체크리스트를 만드세요. ENEX 저장을 끝낼 때마다 체크합니다.

중복 이름은 한 번만 추가됩니다. 체크는 현재 브라우저 탭에서만 유지됩니다.
노트북 목록을 입력하면 빠뜨리지 않고 확인할 수 있습니다.
아직 만든 체크리스트가 없습니다.
    저장 규칙: `업무.enex`, `개인.enex`처럼 노트북 이름으로 같은 `Evernote 전체 백업` 폴더에 저장하세요. 분할 파일도 같은 폴더에 둡니다. 0KB 파일은 그 노트북만 다시 내보냅니다.

    이제 백업 폴더를 딱 한 번 선택하세요

    노트플러스P가 폴더의 ENEX만 골라 저장공간을 점검하고, 저장 전 미리보기에서 파일·노트·첨부·중복·제외 항목을 보여줍니다.

    선택사항입니다. 숫자를 적으면 미리보기에서 발견한 수와 비교해 누락 가능성을 알려드립니다. 휴지통·공유 항목 때문에 차이가 항상 오류를 뜻하지는 않습니다.

    선택 후에도 바로 저장되지 않습니다. 먼저 안전한 미리보기가 열립니다. 고정 용량 제한 없이 큰 ENEX를 파일별로 순차 처리하며, 저장공간이 확실히 부족하면 기존 데이터를 바꾸지 않고 중단합니다.
